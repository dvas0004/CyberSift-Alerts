# Eventlog Cleared
### Description

One of the Windows Eventlogs has been cleared. e.g. caused by "wevtutil cl" command execution.

-------------------
### Severity

- Medium

-------------------

### Detailed Information

May indicate the presence of malware attempting to mask its operation.
